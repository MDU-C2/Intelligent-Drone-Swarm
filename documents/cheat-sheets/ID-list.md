# Project IDs

## ID Rules
- Text first, followed by a hyphen ("-"), and then number.
- Text and numbers are separated by a hyphen to lessen the risk of interpreting I as 1, O as 0, and so on.
- **Example:** ID for goals would be written `G-01`
