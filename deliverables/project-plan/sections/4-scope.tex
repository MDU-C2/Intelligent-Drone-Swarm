\subsection{Assumptions \& Constraints}
%Example: “The website redesign includes new UI/UX, not backend system replacement.”
Since certain specifications require reference to an actual \acrfull{uav}, the team has designated Harris Aerial’s Carrier H6HL \cite{h6hl} as the reference model. This project does not encompass hardware development, and no physical \acrshort{uav} will be available for testing. Instead, validation of the \acrshort{uav} swarm will be conducted through simulations using fault injections.
